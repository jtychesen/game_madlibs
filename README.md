# Mad Libs Game

My first Python project published, later converted (added HTML, CSS and JavaScript) into a playable web app.

## Versions

- Python console version: `main.py`
- Browser version: `index.html`, `style.css`, `script.js`

## Built with

- Python
- HTML
- CSS
- JavaScript
